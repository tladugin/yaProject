# cmd/server

В данной директории будет содержаться код Сервера, который скомпилируется в бинарное приложение.

### Компиляция с информацией о сборке

Для включения информации о версии, дате сборки и коммите используйте флаги `-ldflags`:

#### Сборка Agent с информацией о версии
```bash
# Компиляция агента с передачей информации о версии через ldflags
# -X устанавливает значение переменной во время компоновки
# main.buildVersion - имя пакета.имя_переменной
go build -ldflags="\
    -X 'main.buildVersion=v1.0.0' \           # Версия приложения
    -X 'main.buildDate=$(date +%Y-%m-%d_%H:%M:%S)' \  # Текущая дата и время
    -X 'main.buildCommit=$(git rev-parse HEAD)'" \    # Хэш текущего коммита Git
    -o agent ./cmd/server
